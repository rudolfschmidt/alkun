# alkun
alkun is a http server that use only the api from the java language and is inspirated by expressjs and sinatra but uses custom mechanisms.

## Usage
```xml
<dependencies>
    <dependency>
        <groupId>com.rudolfschmidt</groupId>
        <artifactId>alkun</artifactId>
        <version>3.0</version>
    </dependency>
</dependencies>
```
